# VueBlog

#### 介绍
{**VueBlog博客项目**
前后端分离微服务项目
}

#### 软件架构
前端：
1. Vue
2. Element
3. Node.js
4. axios
后端：

1. SpringBoot
2. SpringCloud Alibaba
3. MyBatis Plus
4. Lombok
5. Logback
6. alibaba-easyexcel
7. Spring Data Redis

#### 环境

1.  jdk 1.8
2.  maven 3.6
3.  ideaIU-2020.2.3
4.  nacos1.4
#### 使用说明

1.  启动前端：文件夹blog-vue，执行npm run dev
2.  启动nacos和redis
3.  启动网关：service-gateway
4.  启动核心业务微服务: service-core
